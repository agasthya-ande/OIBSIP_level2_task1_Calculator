# OIBSIP_level2_task1_Calculator
Description

A simple and attractive calculator developed using HTML, CSS, and JavaScript. The calculator performs basic arithmetic operations such as addition, subtraction, multiplication, and division.

Features

- Addition
- Subtraction
- Multiplication
- Division
- Clear Function
- Responsive User Interface

Technologies Used

- HTML
- CSS
- JavaScript

How to Run

1. Download the project files.
2. Open index.html in any web browser.
3. Start performing calculations.
